# 项目简介 python621

本项目是一个基于 Python Django 和 Vue3 的仓库管理系统案例源码。该系统旨在提供便捷、高效的仓库管理解决方案，支持多角色协同工作，确保仓库运作的顺畅。

## 技术栈

- 后端：Python Django
- 前端：Vue3

## 功能模块

- 用户管理
  - 用户注册与登录
  - 用户权限分配
- 仓库管理
  - 商品信息管理
  - 库存管理
- 订单管理
  - 订单创建与修改
  - 订单追踪

## 系统角色

- 系统管理员
- 仓库管理员
- 普通用户

## 运行环境

- Python 3.8+
- Django 3.x
- Vue3

## 部署步骤

1. 准备 Python 和 Django 的运行环境
2. 安装依赖项：使用 `pip` 安装 `requirements.txt` 文件中的依赖
3. 配置数据库：按照 `settings.py` 中的数据库设置进行配置
4. 运行迁移：执行 `python manage.py migrate` 完成数据库迁移
5. 启动开发服务器：运行 `python manage.py runserver`

## 目录结构

```
project/
├── backend/            # 后端代码目录
│   ├── apps/           # 应用的 Python 包
│   ├── db/             # 数据库相关文件
│   ├── media/          # 媒体文件目录
│   ├── projectname/    # Django 项目设置文件
│   └── static/         # 静态文件目录
└── frontend/           # 前端代码目录
    ├── node_modules/   # Node.js 模块目录
    ├── public/         # 公共文件目录
    └── src/            # Vue.js 源码目录
```

## 核心亮点

- 前后端分离设计，便于开发和维护
- 基于角色权限控制，安全性高
- 采用 Django 的 ORM 系统，数据库操作便捷
- 使用 Vue3 的组合式 API，代码更加模块化和简洁
